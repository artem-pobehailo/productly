

**Productly** — це адаптивна статична посадкова сторінка для продукту, створена за допомогою HTML та CSS.

## Опис

Сайт демонструє сучасний дизайн для презентації продукту або сервісу. Він містить:
- хедер із навігацією та кнопками `Sign In` / `Sign Up`
- секцію `hero` із заголовком, описом, кнопками та ілюстрацією
- блоки з описом функціональних переваг для дизайнерів, продукт-менеджерів і маркетологів
- футер із додатковими посиланнями та формою підписки

## Стек

- HTML
- CSS
- Статичні зображення та SVG-іконки

## Структура проєкту

- `index.html` — основна сторінка
- `styles.css` — стилі для дизайну сторінки
- `script.js` — порожній файл для потенційного JavaScript
- `images/` — директорія з логотипами, ілюстраціями та SVG-спрайтом
- `prettierrc.json` — налаштування форматування

## Як запустити

1. Відкрийте `index.html` у веб-браузері.
2. Або скористайтесь локальним сервером (наприклад, Live Server у VS Code).

## Можливі розширення

- додати адаптивну верстку для мобільних пристроїв
- додати JavaScript для інтерактивності
- реалізувати форму підписки з валідацією
- підключити реальні посилання у навігації та футері

## Примітка

Наразі сайт є макетом посадкової сторінки і працює без складної логіки.
